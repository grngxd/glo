## usage
`glo [-padding 30] [-master 0.6]`

## hotkeys
- Win+Shift+O: toggle tiling
- Win+Shift+=: grow master
- Win+Shift+-: shrink master
- Win+Shift+.: rotate master
- Win+Shift+Q: quit
